# Unit Test

Unit testing merupakan pengujian perangkat lunak yang menguji setiap unit atau komponen dari sebuah perangkat lunak. Unit testing dilakukan saat masa pengembangan (menuliskan kode program) dari sebuah perangkat lunak oleh si pengembang.

Pada web application kopmart, dilakukan unit testing dengan menggunakan framework [JUnit](https://junit.org/junit4/) dan [Mockito](https://site.mockito.org/) pada layer service.

```java
package com.deta.kopmart_backend;

import com.deta.kopmart_backend.service.impl.*;
import org.junit.runner.RunWith;
import org.junit.runners.Suite;

@RunWith(Suite.class)
@Suite.SuiteClasses({
		CartServiceImplTest.class,
		CategoryServiceImplTest.class,
		OrderServiceImplTest.class,
		ProductInOrderServiceImplTest.class,
		ProductServiceImplTest.class,
		UserServiceImplTest.class
})
class KopmartBackendApplicationTests {

}
```
